# Translate Pollaris

**You can help to translate Pollaris at [weblate.framasoft.org](https://weblate.framasoft.org/engage/pollaris/).**

[![Translation status](https://weblate.framasoft.org/widget/pollaris/-/en/svg-badge.svg)](https://weblate.framasoft.org/engage/pollaris/)

## Adding a new translation

You can submit a new translation directly on Weblate.
Once you have submitted it, a maintainer will accept your translation, which should then become available in the next release.

**Note to the maintainers:** when accepting a new translation, you must:

- add the language to the supported languages in [`src/Utils/Locales.php`](/src/Utils/Locales.php);
- enable the language in [`assets/js/dayjs_locale.js`](/assets/js/dayjs_locale.js) ([see supported locales](https://github.com/iamkun/dayjs/tree/dev/src/locale)).
